# 🍔 Food Delivery Time Analysis

## 📌 Problem Statement
The project aims to analyze food delivery data to understand the key factors affecting delivery time and identify ways to improve efficiency.

---

## 🎯 Objective
To perform Exploratory Data Analysis (EDA) on the food delivery dataset and identify patterns related to delivery delays.

---

## 📊 Dataset
- Source: Food Delivery Dataset (CSV)
- Features:
  - Distance (km)
  - Weather conditions
  - Traffic level
  - Courier experience (years)
  - Preparation time (minutes)
  - Delivery time (minutes)

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔍 Process
1. Data Cleaning (handled missing values & duplicates)
2. Data Analysis using Pandas
3. Visualization using charts
4. Identified key insights affecting delivery time

---

## 📈 Visualizations Used
- Bar Charts → To compare average delivery time across categories
- Histogram → To understand delivery time distribution
- Scatter Plot → To analyze relationship between distance and delivery time
- Line Plot → To study courier experience impact
- Heatmap → To find correlation between variables

---

## 💡 Key Insights
- Delivery time increases with distance
- Bad weather conditions increase delivery delays
- High traffic leads to longer delivery times
- Courier experience has slight impact on delivery efficiency

---

## 🚀 Conclusion
This project helps understand the factors affecting delivery time and can be used by companies to optimize delivery operations.
